# Анализ геополитических нарративов LLM

Этот проект посвящен исследованию ответов различных больших языковых моделей (LLM) на геополитические запросы, связанные с санкциями, технологическим противостоянием и региональными конфликтами.

## Структура проекта

Репозиторий организован следующим образом:

- **`data/`**: Наборы данных в формате CSV.
  - `module1/`: Данные по санкциям и таймлайнам.
  - `module2/`: Результаты опросов моделей.
  - `module3/`: Классифицированные ответы и статистика (основной модуль анализа).
- **`src/`**: Исходный код на Python.
  - `module3/`: Скрипты классификации, визуализации и аудита данных.
- **`output/`**: Итоговые визуализации и графические отчеты.
- **`docs/`**: Документация и методология исследования.

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url-репозитория>
   cd Geopolitical_LLM_Analysis
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения:**
   Скопируйте файл `.env.example` в `.env` и добавьте свои ключи API (например, Gemini API):
   ```bash
   cp .env.example .env
   ```

4. **Запустите визуализацию:**
   ```bash
   python src/module3/visualize.py
   ```

## Используемые данные
